git commands:
	git init：初始化本地库
	git add 文件名 将文件添加到暂存区
	git commit -m "text" 文件名 提交到本地库
	git status 查看本地库状态
	git reflog 查看历史版本信息
	git log 详细查看历史版本信息
	
	版本穿梭:
		git reset --hard 版本号
		通过head指针指向某个分支的某个版本

	git branch -v 查看分支
	git branch 分支名 创建分支
	git checkout 分支名 切换分支

	git merge 分支名 将分支名合并到当前所在的分支
                1、<<<<到====是当前分支的内容 2、====到>>>>是被合并分支的内容
		2、提交时不要加上文件名

	git remote -v 查看是否有别名
	git remote add 远程库地址(别名) 分支名

	git push 远程库名 分支名 把本地库的某个分支推送到远程库
	git pull 远程库名 分支名 把远程库拉取到本地库
	
	git clone 远程库地址 克隆远程库(不需要登陆github账号)
